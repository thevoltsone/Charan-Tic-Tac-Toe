# Premium Tic Tac Toe

A beautifully designed, interactive Tic Tac Toe web application featuring a modern aesthetic with glassmorphism, dynamic text gradients, and smooth user interactions.

## Features

- **Sleek UI/UX:** Clean, modern dark mode design with vibrant gradient colors.
- **Glassmorphism:** Elegant semi-transparent elements with blur effects.
- **Smooth Animations:** Cell pop-in animations, visual hover states, and stylized winner overlays.
- **Responsive Design:** Optimized for both desktop and mobile screens.
- **Accessibility Checked:** Basic keyboard navigation (Space/Enter) and screen reader support elements.

## How to Play

1. Open `index.html` in your favorite web browser.
2. Player X always starts first.
3. Click on any empty cell (or use 'Tab' and 'Enter' as a keyboard alternative) to place your mark.
4. The first player to get 3 marks in a row (horizontally, vertically, or diagonally) wins.
5. If all cells are filled and no player wins, the game ends in a draw.
6. Click "Play Again" or "Reset Game" to start a new round.

## Installation / Setup

No complicated build tools required! To run this project locally:

1. Clone the repository: 
   ```bash
   git clone https://github.com/yourusername/tic-tac-toe.git
   ```
2. Navigate to the project directory:
   ```bash
   cd tic-tac-toe
   ```
3. Open `index.html` in your web browser.
